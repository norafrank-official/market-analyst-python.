An advanced financial tracking tool built to demonstrate full-stack Python capabilities. This application processes live market data, handles complex data flattening, and applies quantitative logic to generate real-time trading sentiment.

---

## 🚀 Key Functionalities

| Feature | Description |
| :--- | :--- |
| **Real-Time Streaming** | Asynchronous integration with Yahoo Finance for 24/7 price tracking. |
| **Signal Engine** | Automated Buy/Sell/Hold recommendations based on RSI & SMA crossover logic. |
| **Data Analytics** | Automated flattening of MultiIndex DataFrames and rolling window calculations. |
| **Dynamic UI** | Interactive Plotly candlestick charts with synchronized technical overlays. |

## 🛠️ Technical Implementation
* **RSI (Relative Strength Index):** Implemented a 14-period momentum oscillator to identify overbought/oversold conditions.
* **SMA (Simple Moving Average):** 20-period trend analysis to determine market direction.
* **Streamlit Caching:** Optimized API calls using `@st.cache_data` to ensure high performance and low latency.
